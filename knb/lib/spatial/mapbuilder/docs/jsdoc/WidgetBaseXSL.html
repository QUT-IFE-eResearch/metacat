WidgetBaseXSL

WidgetBaseXSL(widgetNode,model)
    Base Class for widgets. Associates a node on the page with a stylesheet and model. All widgets must extend this base class. Defines the default paint() method for all widgets which is where the stylesheet is applied to the model XML document. To override widget.paint(), define it before calling this constructor. The stylesheet URL defaults to "widget/.xsl" if it is not defined in config file. Set a stylesheet property containing an XSL URL in config to customize the stylesheet used. All stylesheets will have "modelId" and "widgetId" parameters set when called.
    Parameters:
      widgetNode - The widget's XML object node from the configuration document.
      model - The model object that this widget belongs to.
      widget - Pointer to the widget instance being created
    Author: Mike Adair
