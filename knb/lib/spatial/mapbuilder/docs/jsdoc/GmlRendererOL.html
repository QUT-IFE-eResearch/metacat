Constructor Detail

GmlRendererOL

GmlRendererOL(widgetNode, model)
    Render GML into HTML. this.targetModel references the context model for the map where the content of this widget should be rendered to. If the model doc is not wfs compliant, a stylesheet property has to be set for this widget. The xsl file referenced in this property transforms the model doc to a wfs FeatureCollection.
    Parameters:
      widgetNode - The widget's XML object node from the configuration document.
      model - The model object that this widget belongs to.
