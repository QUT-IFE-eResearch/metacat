CollectionList

CollectionList(widgetNode, model)
    A widget to display a list of context docs to pick from. This is a view of a Context Collection as specified in the OGC Context specification. The default xsl stylesheet for this widget also uses the switchMap function, which allows to switch to a different map context.
    Parameters:
      widgetNode - The widget's XML object node from the configuration document.
      model - The model object that this widget belongs to.
